# ChamuyoCheck V8 - Experiencia Premium

Cambios principales:
- Se elimina la lógica de múltiples botones principales.
- Un único cuadro inteligente: arrastrar, pegar o escribir.
- Detección automática de texto, PDF, imagen, web o YouTube.
- Área de carga mucho más visible y clara.
- Indicadores circulares y barras de riesgo.
- Starter: 3 análisis, texto hasta 250 caracteres.
- Pro demo: desbloquea PDF, imagen, web y YouTube.
- El análisis sigue siendo prudente: no acusa, no determina mentira, estafa, plagio o diagnóstico.

Nota técnica:
V8 mejora la experiencia. La lectura real de PDF/OCR/web/YouTube puede profundizarse en V8.1/V9 con servicios específicos.
