# ChamuyoCheck V4

Versión 4 del MVP: **Sistema Inteligente de Auditoría de Credibilidad**.

## Qué agrega

- Orquestador de análisis por módulos.
- Clasificación automática del tipo de texto.
- Módulo financiero-matemático integrado.
- Radar de esquemas piramidales / Ponzi / referidos.
- Resguardo legal: lenguaje prudente, sin acusaciones categóricas.
- Peor escenario razonable.
- Información faltante.
- Comité de análisis: evidencia, psicología, promesas, finanzas y radar piramidal.

## Regla central

ChamuyoCheck **no afirma que alguien miente**. Evalúa señales de riesgo, respaldo disponible, información faltante y preguntas que conviene hacer antes de decidir.

## Deploy

Copiar el contenido de esta carpeta al repositorio local con GitHub Desktop, hacer commit y push. Vercel redeploya automáticamente.
