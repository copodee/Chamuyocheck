# CHAMUYOCHECK V15.13 -- UX, Context Intelligence & Smart Reports

## Objetivo

Esta versión transforma ChamuyoCheck desde un sistema que muestra
resultados hacia una plataforma que explica, contextualiza y acompaña la
toma de decisiones.

Se recupera la comunicación comercial del producto, se corrigen
inconsistencias de UX detectadas durante las pruebas y se implementa
inteligencia contextual para que cada informe se adapte automáticamente
al tipo de contenido analizado.

------------------------------------------------------------------------

# Objetivos

-   Recuperar el Hero comercial perdido.
-   Unificar la experiencia Desktop y Mobile.
-   Eliminar respuestas genéricas.
-   Adaptar dinámicamente el informe al contexto.
-   Corregir bugs funcionales.
-   Mejorar la comprensión del puntaje.
-   Aumentar la percepción de inteligencia del producto.

------------------------------------------------------------------------

# 1. Hero Comercial (Desktop)

Recuperar el bloque introductorio ubicado antes del formulario.

Debe explicar claramente qué hace ChamuyoCheck y generar confianza desde
el primer segundo.

## Título

**Analizá antes de decidir.**

## Bajada

No todo lo que leés, mirás o te prometen merece confianza.

ChamuyoCheck analiza textos, documentos, imágenes, páginas web y videos
para ayudarte a detectar señales de manipulación, falta de evidencia,
riesgos potenciales y aspectos que conviene verificar antes de tomar una
decisión.

Subí un PDF, pegá un texto o una URL y obtené un informe claro en
segundos.

### Beneficios

-   Detecta señales de manipulación.
-   Evalúa credibilidad y evidencia.
-   Resume riesgos importantes.
-   Sugiere qué verificar antes de decidir.
